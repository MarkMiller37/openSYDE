Description:
============
SYDEsup is a console application for updating your system with a Service
Update Package created with openSYDE GUI.

Features:
* Unpack Service Update Package and update system with resulting information
* Communication via CAN or Ethernet
* Configuration through command line parameters (run SYDEsup without parameters to get a list of options)
* Create Service Update Package from a specific view of a given openSYDE project and corresponding device definitions

Non-Features:
* Update system without previous use of openSYDE or SYDEsup to create Service Update Package

Platform:
=========
MS Windows (Win10 verified)
Ubuntu Linux (Ubuntu 20.04.1 LTS verified)

Requires:
=========
- Windows: STW "CANDLL" to access CAN bus
